Compress the cloud data.

This function compresses the PointCloud data using the pcl::PointCloudCompression<pcl::PointXYZ> class. Returns the compressed data in he form of std::stringstream. The compression profile is pcl::octree::LOW_RES_ONLINE_COMPRESSION_WITHOUT_COLOR
